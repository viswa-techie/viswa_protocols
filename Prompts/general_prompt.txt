# 1. Basic Communication Classification

| Type             | Meaning                           |
| ---------------- | --------------------------------- |
| Serial           | 1 bit at a time                   |
| Parallel         | Multiple bits simultaneously      |
| Synchronous      | Uses shared clock                 |
| Asynchronous     | No shared clock                   |
| Half Duplex      | Both directions but one at a time |
| Full Duplex      | Both directions simultaneously    |
| Point-to-Point   | One device ↔ one device           |
| Multi-drop / Bus | Multiple devices share bus        |
| Wired            | Physical medium                   |
| Wireless         | RF/IR/Optical                     |
