STANDARDS UNIVERSE:
SYSTEMS ENGINEERING LIFECYCLE:
  ISO/IEC/IEEE 15288:2023       — System and Software Engineering: System Lifecycle Processes
      Technical processes: stakeholder needs definition, requirements, architecture, implementation,
                            integration, verification, transition, validation, operation, maintenance
      Management processes: project, quality, risk, configuration, information, measurement
      Agreement processes: acquisition, supply
  ISO/IEC/IEEE 12207:2017       — Software Lifecycle Processes (companion to 15288)
  ISO/IEC 29148:2018            — Requirements Engineering: stakeholder, system, software requirements
  ISO/IEC 33001:2015            — Process Assessment: concepts and terminology
  IEEE 1012:2016                — Verification and Validation (V&V)
  IEEE 1016:2009                — Software Design Descriptions (SDD)
  IEEE 830 (superseded)         — Software Requirements Specifications (SRS) format

MBSE (MODEL-BASED SYSTEMS ENGINEERING):
  SysML v1.6 (OMG, 2019)       — Systems Modeling Language: BDD, IBD, REQ, PAR, ACT, SEQ, STM, UC
  SysML v2 (OMG, 2024)         — Major revision: KerML kernel; structured expressions; improved semantics
  Harmony for SysML (IBM)       — Top-down design methodology (ROPES predecessor)
  IBM Rhapsody                   — SysML/UML MBSE tool
  Capella (Arcadia)              — Open source MBSE tool (Thales methodology)
  PTC Windchill/Integrity       — PLM integration with MBSE
  Modelica                      — Physical system simulation language
  MBSE adoption: NASA, ESA, Airbus, Boeing, Siemens, German automotive OEMs

HISTORICAL TIMELINE:
  1957 — Systems Analysis (RAND Corporation); PERT network (Polaris program)
  1960 — SAGE air defense: first large-scale software system
  1968 — NATO Software Engineering conference (software crisis)
  1969 — Apollo 11: concurrent systems engineering, V&V processes
  1988 — IEEE 1012: V&V standard
  1994 — SEI Capability Maturity Model (CMM); early SE maturity
  1998 — ISO/IEC 15288 development begins; INCOSE
  2000 — INCOSE Systems Engineering Handbook v1
  2003 — SysML 1.0 (OMG); MBSE concept popularized
  2005 — ISO/IEC/IEEE 15288:2008 first edition
  2010 — INCOSE MBSE Challenge; Capella tool (Thales)
  2015 — INCOSE SE Handbook v4; SysML 1.4
  2017 — ISO/IEC 15288:2023 revision begins
  2019 — SysML v2 alpha; OMG formal submission
  2023 — ISO/IEC/IEEE 15288:2023; SysML v2 beta
  2024 — SysML v2 adopted by OMG; Arcadia/Capella v7
