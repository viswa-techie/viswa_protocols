FIDO ALLIANCE:
  FIDO2 = WebAuthn + CTAP2      — Passwordless authentication standard (2018)
  W3C WebAuthn Level 3          — Web Authentication API specification
  FIDO CTAP2.2                  — Client to Authenticator Protocol (hardware keys/biometric)
  FIDO UAF 1.1                  — Universal Authentication Framework (mobile biometric)
  FIDO U2F 1.2                  — Universal 2nd Factor (legacy hardware token)
  FIDO Passkeys                 — Syncable passkeys (Apple/Google/Microsoft — 2022)
  FIDO Device Onboard (FDO)     — IoT device onboarding specification
  FIDO Alliance Certification   — L1/L2/L3 authenticator certification levels

TRAVEL DOCUMENTS (e-PASSPORT):

      Part 9: Deployment of biometric identification and electronic storage of data
      Part 10: Logical data structure for the storage of data
      Part 11: Security mechanisms for MRTDs
  ICAO PKD                      — Public Key Directory for e-passport verification
  BAC (Basic Access Control)    — Passive authentication for RFID chip

  EAC (Extended Access Control) — For fingerprints and iris (restricted biometrics)
  AA (Active Authentication)    — Anti-cloning of e-passport chip
  SAC (Supplemental Access Control) — PACE replacement for BAC

HISTORICAL CONTEXT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1858 — William Herschel uses fingerprints for identification in India (colonial admin)
1892 — Juan Vucetich: first criminal conviction by fingerprint (Argentina)
1903 — New York State Prison adopts fingerprint identification (USA)
1968 — Roland Moreno invents smart card concept (patent filed 1974)
1994 — SET (Secure Electronic Transactions): first attempt at digital identity
         FIPS PUB 201 PIV card concept begins
2000 — ICAO Doc 9303 updated for digital travel documents
2001 — Post-9/11: US VISIT program (biometric entry); ICAO e-passport mandate push
2004 — First e-passports issued (Malaysia, Belgium, USA)
2005 — NIST SP 800-63 v1: first authentication assurance guidelines
         EU eIDAS predecessor (Directive 1999/93/EC electronic signatures)
2013 — FIDO Alliance founded (Google, PayPal, Lenovo, NXP, Nok Nok Labs)
2014 — FIDO UAF 1.0 and U2F 1.0 published
         eIDAS Regulation 910/2014 published
2016 — NIST SP 800-63-3: identity assurance levels (IAL/AAL/FAL) framework
2018 — W3C WebAuthn published; FIDO2 launched
         GDPR enforcement: biometric data as "special category" personal data
2021 — ICAO LDS2 (e-passport new applications: health, additional biometrics)
2022 — Passkeys announced (Apple WWDC, Google I/O, Microsoft) — mass market FIDO2
2023 — Apple/Google/Microsoft passkey implementation ships to billions of devices
         eIDAS 2.0 political agreement reached
2024 — eIDAS 2.0 / EUDIW Regulation published; 24-month implementation clock starts
         NIST SP 800-63-4 second public draft
2025 — EU member states deploying national EUDIW wallets
         Post-quantum considerations entering ICAO e-passport discussion
