STANDARDS UNIVERSE:
DATA CENTER FACILITY:
  Uptime Institute Tier Standards — Tier I (99.671%), II (99.741%), III (99.982%), IV (99.995%) availability
      Tier I: Non-redundant power/cooling; single path
      Tier II: Redundant components; single path
      Tier III: N+1 redundancy; concurrent maintainability; no shutdowns for maintenance
      Tier IV: 2N or 2N+1 redundancy; fault tolerant; 96 hours power outage tolerance
  ANSI/BICSI 002-2019           — Data Center Design and Implementation Best Practices
  EN 50600 series (CENELEC)     — Information technology: Data center facilities
      EN 50600-1: General concepts
      EN 50600-2-1: Building construction
      EN 50600-2-2: Power distribution
      EN 50600-2-3: Environmental control
      EN 50600-2-4: Telecommunications cabling
      EN 50600-3-1: Management and operational information

ENERGY EFFICIENCY METRICS:
  PUE (Power Usage Effectiveness) — The Green Grid: total DC power / IT power; target < 1.3
  WUE (Water Usage Effectiveness) — Annual water / IT energy; Google/MS report WUE
  CUE (Carbon Usage Effectiveness) — Carbon footprint per IT power
  ERE (Energy Reuse Effectiveness) — Waste heat reuse
  ASHRAE TC 9.9                  — Thermal guidelines for data processing environments
      Class A1-A4 (server) temperature and humidity envelopes
  ISO/IEC 30134 series           — Data center key performance indicators
  EU CoC Data Centres (2023)    — European Code of Conduct for DC energy efficiency

OPEN COMPUTE PROJECT (OCP):
  OCP Open Rack v3              — Server rack specification (19"/21" vs. OCP)
  OCP Server (Leopard, Yosemite V3) — Facebook/Meta hyperscale server standards
  OCP NIC 3.0                   — Network interface standards
  OCP Storage                    — Storage shelf specifications
  OCP OSFP/QSFP-DD               — Optical transceiver form factors (400G/800G)

CLOUD SECURITY:
  ISO/IEC 27017:2015            — Cloud security (control guidance on ISO 27002)
  ISO/IEC 27018:2019            — PII protection in public cloud environments
  CSA STAR (Security Trust Assurance and Risk) — Cloud provider self-assessment
      STAR Level 1: Self-assessment (CAIQ questionnaire)
      STAR Level 2: Third-party audit (STAR Certification ISO 27001 + CCM)
      STAR Level 3: Continuous monitoring
  CSA CCM v4.0 (Cloud Controls Matrix) — 197 control objectives; 17 domains
  FedRAMP (US Federal)          — Federal Risk and Authorization Management Program
      Baselines: Low, Moderate, High (based on NIST SP 800-53)
      Authorization paths: Agency ATO, JAB P-ATO
  EUCS (EU Cloud Scheme)        — ENISA Cloud Certification Scheme (Basic, Substantial, High)
  C5 (Germany BSI)              — Cloud Computing Compliance Criteria Catalog

CLOUD SERVICE PROVIDER COMPLIANCE:
  AWS: SOC 1/2/3, PCI DSS, ISO 27001, FedRAMP, HIPAA BAA
  Azure: SOC 1/2/3, ISO 27001/27017/27018, FedRAMP High, DISA IL2-6
  GCP: SOC 1/2/3, ISO 27001, FedRAMP, HIPAA, PCI DSS Level 1

NETWORKING STANDARDS (DATA CENTER):
  IEEE 802.3bs (400G Ethernet)   — 400GBASE-SR16, -LR8, -FR8 (2017)
  IEEE 802.3ck (100G/400G/800G)  — 112 Gbps lanes (2022)
  IEEE 802.3df (800GbE)          — 800 Gbps Ethernet (2024)
  SNIA SNIA-TW-2001              — Snia Twiki storage interface
  NVMe over Fabrics (NVMe-oF)    — NVMe/RDMA (RoCE), NVMe/FC, NVMe/TCP
  InfiniBand (IBTA HDR 200Gbps, NDR 400Gbps) — HPC and AI cluster interconnect
  RDMA (iWARP, RoCE v2)         — Remote Direct Memory Access
  Ultra Ethernet Consortium (UEC) — AI scale-out network (see Cat 40)

CONTAINER & KUBERNETES STANDARDS:
  OCI (Open Container Initiative) — Runtime Spec, Image Spec, Distribution Spec
  Kubernetes SIG Security        — Pod Security Admission; RBAC; secrets management
  CIS Kubernetes Benchmark       — Hardening guide (kube-apiserver, kubelet, etcd)
  CNCF Security WG               — Cloud native security whitepaper
  SLSA (Supply Chain Levels for Software Artifacts) — see Category 36

HISTORICAL TIMELINE:
  1996 — First commercial data centers; Tier concept emerging
  2000 — Uptime Institute Tier Standard codified
  2005 — Virtualization boom (VMware ESX 3.0)
  2006 — AWS EC2 (cloud computing era); Amazon S3
  2008 — The Green Grid: PUE metric introduced
  2009 — Google: first public PUE reporting; OCP beginning
  2011 — Facebook Open Compute Project launched
  2012 — ASHRAE A1-A4 server classification; cloud bursting models
  2015 — EU Code of Conduct data centers; ISO/IEC 27017
  2019 — FedRAMP Moderate widespread adoption
  2020 — COVID: hyperscale DC expansion; OCP expansion
  2021 — EU data localization requirements; CSA CCM v4.0
  2022 — OCP OAI; ultra-low latency DC for fintech
  2023 — EUCS scheme draft; AI cluster DC power density >100kW/rack
  2024 — AI DC power crisis: NVIDIA H100/H200 racks; liquid cooling mandates
