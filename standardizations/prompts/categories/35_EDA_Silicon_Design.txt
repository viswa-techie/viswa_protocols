HARDWARE DESCRIPTION & VERIFICATION LANGUAGES:
  IEEE 1800-2023               — SystemVerilog (unified HDL + HVL)
      Synthesizable subset (RTL) and verification subset (OOP classes, interfaces, clocking)
  IEEE 1364-2001               — Verilog (legacy; subsumed by 1800-2012)
  IEEE 1076-2019               — VHDL (VHSIC Hardware Description Language)
  IEEE 1666-2011               — SystemC (system-level modeling, TLM 2.0)
  Accellera PSS (Portable Stimulus Standard) 2.0 — Portable verification intent
  SVA (SystemVerilog Assertions) — Concurrent + immediate assertions, sequences, properties
  e Language (Specman)         — Cadence e verification language (IEC 62531:2007)

VERIFICATION METHODOLOGY:
  Accellera UVM 1.2 (2014)     — Universal Verification Methodology (gold standard)
  Accellera UVM 2017 (EA)      — Evolution of UVM (class library, messaging)
  VMM (Synopsys)               — Verification Methodology Manual (pre-UVM)
  OVM (Cadence/Mentor)         — Open Verification Methodology (pre-UVM)
  Formal Verification:
      IEEE P2860                — IEEE Standard for Formal Methods
      PSL (Property Specification Language) — IEEE 1850-2010
      ABV (Assertion-Based Verification) — Synopsys VC Formal, Cadence JasperGold
  Functional Coverage:
      UVM Coverage API          — Covergroup, coverpoint, cross coverage
      ISO 26262 Part 6 Clause 11 — Coverage requirements for ASIL

POWER INTENT STANDARDS:
  IEEE 1801-2023               — Unified Power Format (UPF) — primary standard
  CPF (Common Power Format)    — Cadence legacy (deprecated, UPF superseded)
  Low-Power Verification:
      LPDDR power management interface (JEDEC JESD79F)
      ARM AMBApower signaling protocol (Q-Channel, P-Channel)

DESIGN EXCHANGE & IP:
  IEEE 1685-2014               — IP-XACT (XML schema for IP reuse)
  IEEE 1687-2014               — iJTAG (Internal JTAG for instrument integration)
  IEEE 1838-2019               — Die, Stack, and System Test (3D stacked ICs)
  IEEE 1500-2005               — Wrapper for Embedded Core Test (SECT)
  Accellera TLM 2.0            — Transaction-Level Modeling (SystemC)

TIMING & LIBRARIES:
  Liberty (.lib) Format        — Synopsys Liberty syntax for cell timing/power
  Timing Constraints:
      SDC (Synopsys Design Constraints) — De facto standard for STA
      MMMC (Multi-Mode Multi-Corner)    — Industry practice for STA
  SDF (Standard Delay Format) — IEEE 1497 — Back-annotation
  SPEF (Standard Parasitic Exchange Format) — IEEE 1481 — RC extraction
  DSPF (Detailed Standard Parasitic Format) — Cadence detailed RC

PHYSICAL DESIGN:
  DEF (Design Exchange Format) — Cadence/Synopsys physical layout
  LEF (Library Exchange Format) — Standard cell characterization layout
  GDS II / GDSII               — Graphic Design System (layout export)
  OASIS (Open Artwork System Interchange Standard) — Replace GDSII for advanced nodes
  SKILL (Cadence) / Python OpenAccess — Database APIs
  OpenDB / OpenROAD            — Open-source physical design tools (DARPA initiative)

DFT (DESIGN FOR TEST):
  IEEE 1149.1-2013             — Standard Test Access Port (JTAG boundary scan)
  IEEE 1149.6-2003             — Differential and AC-coupled JTAG
  IEEE 1149.7-2009             — Reduced-pin and enhanced JTAG
  IEEE 1500-2005               — Embedded core test wrapper
  IEEE 1687-2014               — iJTAG reconfigurable scan networks
  IEEE P1838                   — 3D stacked IC DFT (die-to-die)
  ATPG (Automatic Test Pattern Generation):
      Fault models: Stuck-at, Transition, Path-Delay, Cell-Aware
      Industry tools: Synopsys TetraMAX, Cadence Modus

FORMAL & STATIC ANALYSIS:

HISTORICAL CONTEXT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1964 — Gordon Moore's Law observation (doubling transistors every ~2 years)
1975 — SPICE circuit simulator (UCB): foundation of EDA simulation
1978 — First VLSI conference; HDL concepts emerging
1984 — Verilog invented at Gateway Design Automation (Phil Moorby)
1987 — VHDL 1987 published (IEEE 1076-1987): US DoD requirement
1990 — Synopsys Design Compiler: first commercial synthesis tool
1995 — Verilog standardized as IEEE 1364-1995
         OVM/VMM verification methodology concepts emerge
2001 — SystemVerilog 3.0 (Accellera) — merged HDL + HVL
2005 — SystemVerilog IEEE 1800-2005 first standardization
         UPF (CPF + UPF merge ongoing)
2006 — e Language standardized as IEC 62531
2009 — IEEE 1800-2009 (major SystemVerilog revision)
         OVM becomes open, precursor to UVM
2011 — UVM 1.0 released by Accellera (grand unification of VMM + OVM)
         IEEE 1801-2009 UPF (unified power intent)
2012 — IEEE 1800-2012 (merged Verilog+SV into one standard)
2014 — UVM 1.2 (current widely-used version); IP-XACT IEEE 1685
2015 — FinFET adoption (16/14nm): power/verification methodology challenges
2017 — IEEE 1800-2017 revision; PSS 1.0 (Accellera Portable Stimulus)
2020 — DARPA Open-Source EDA (OSDA): OpenROAD, OpenLane project
         SkyWater 130nm first open PDK with Google
2023 — IEEE 1800-2023 (latest SV); AI-assisted EDA mainstream (Synopsys DSO.ai)
2024 — Gate-all-around (GAAFet/MBCFET) at 2nm/3nm process; new PDK requirements
         Chiplet design (UCIe): multi-die EDA methodology
