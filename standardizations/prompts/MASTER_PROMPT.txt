This Master Prompt System is a collection of 25 deep-dive category prompts + this
orchestration master. Each prompt in the categories/ folder is designed to independently
generate a complete, book-quality engineering compliance guide for that domain.

PURPOSE:
  This system enables you to generate a professional-grade TECHNOLOGY STANDARDIZATION
  ENCYCLOPEDIA equivalent to what is maintained internally at:
    - Bosch, Continental, Aptiv, Harman (Automotive Tier-1)
    - Qualcomm, Intel, NVIDIA, NXP, Renesas (Semiconductor)
    - Airbus, Boeing, Lockheed Martin (Aerospace)
    - Google, AWS, Microsoft (Cloud/AI)
    - Medtronic, Siemens Healthineers (Medical)
    - ISO, IEC, IEEE, NIST, SAE (Standards Bodies)

HOW TO USE:
  Step 1: Use THIS master prompt to generate the root-level README, index, and
          navigation architecture for the entire knowledge base.
  Step 2: Use each category prompt (01 through 25) to generate a complete multi-chapter
          guide for that domain. Each generates a standalone book.
  Step 3: The output repository maps 1:1 to the folder structure defined in Section 2.
  Step 4: Cross-reference guide (generated from Section 6) links standards across books.

FILE NAMING CONVENTION:
  categories/01_Functional_Safety_prompt.txt      → generates: 01_Functional_Safety/
  categories/02_Automotive_Standards_prompt.txt   → generates: 02_Automotive_Standards/
  ... and so on for all 25 categories.
